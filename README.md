# offsites-rec

# How to run

Create a .env file to hold the API_KEY:
```
API_KEY=<your-api-key>
```

Run the script to get the flight averages and total price:
```
npm run getFlightEstimations
```

Run the script to get the averages for one flight:
```
npm run getFlight
```

# Tests

```
npm run test
```
